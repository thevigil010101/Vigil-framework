Psychedelic Medicine & VC

Using medicines as temporary VC expansion tools

CRITICAL SAFETY WARNING

DO NOT use psychedelics if your VC is below 45.

Low VC + High wave = Overwhelm, trauma, psychosis risk.

Build beach FIRST. Then add bigger waves.

What Medicines Do

Psychedelics = Temporary VC boost

They don't CREATE capacity. They temporarily EXPAND access to substrate.

Like widening your browser bandwidth - more data flows through, but your computer (brain) still needs to process it.

Medicine-Specific Effects

Psilocybin (Mushrooms)

Temporary VC boost: +20 to +40 points

Duration: 4-6 hours

Best for: Emotional processing, nature connection, gentle integration

Minimum baseline: VC = 45

LSD

Temporary VC boost: +30 to +50 points

Duration: 8-12 hours

Best for: Pattern recognition, cognitive restructuring, long integration

Minimum baseline: VC = 50

DMT

Temporary VC boost: +50 to +700 points (breakthrough)

Duration: 5-15 minutes

Best for: Substrate perception, entity contact, paradigm shifts

Minimum baseline: VC = 60 (DO NOT attempt below this)

MDMA

Temporary VC boost: +15 to +25 points

Duration: 4-6 hours

Best for: Trauma processing, emotional connection, therapeutic work

Minimum baseline: VC = 40

The Integration Rule

The medicine experience isn't the work. Integration is the work.

After Medicine:

  1. Day 1-3: Rest. Let experience settle. Gentle journaling.
  2. Week 1: Deep journaling. What did you see? What needs processing?
  3. Week 2-4: Active integration. Apply insights. Change behaviors.
  4. Month 2-3: Solidify changes. Let new patterns cement (good cement, not suppression).
  5. Month 6+: Consider next medicine session IF baseline VC has risen.

DO NOT use medicine frequently. 3-6 month minimum between sessions.

Frequent use = Spiritual bypassing, not integration.

Set & Setting

SET = Your internal state (VC, intentions, readiness)

SETTING = Your external environment (physical safety, emotional safety)

Required Before Medicine:

  • ✓ Baseline VC above minimum for that medicine
  • ✓ Physically safe location
  • ✓ Trusted sitter/guide present
  • ✓ No obligations for 48 hours after
  • ✓ Integration plan prepared
  • ✓ Emergency resources available
  • ✓ Honest intention (not escaping, exploring)

When NOT To Use Medicine

  • VC below minimum for that medicine
  • Active psychosis or schizophrenia
  • Severe depression (below VC = 25)
  • Suicidal ideation present
  • Using to escape vs. integrate
  • No integration plan
  • Unsafe setting

The Honest Truth

Medicine can be powerful. Medicine can also traumatize if misused.

Build your beach FIRST.

Then, when ready, medicines can show you what's possible.

But you still have to do the work to stay there.

Medicine shows what's possible. Integration makes it permanent.

Build beach first. Add medicine later. Always integrate after.
